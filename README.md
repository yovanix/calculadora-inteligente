Este projeto consiste em uma calculadora inteligente que utiliza lógica básica de JavaScript para realizar cálculos simples como adição, subtração, divisão e multiplicação. Ela também está configurada para reconhecer teclas do computador e registrar o histórico de cálculos anteriores.
